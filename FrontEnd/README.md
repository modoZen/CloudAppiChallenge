# Cloudappi-Front End
Este codigo corresponde un SPA CRUD con Vue js

## Configuracion del proyecto
```
npm install
```

### Compila y recarga en caliente para el desarrollo
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
